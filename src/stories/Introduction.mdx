## Best Practices

### Using Components

1. **Import from the correct path**:
   ```tsx
   // Base components
   import { Button } from '@/components/ui/Button';
   
   // Feature components
   import { GameCard } from '@/components/features/GameCard';
   ```

2. **Follow composition patterns**:
   ```tsx
   // Compose feature components from base
   <Card>
     <CardHeader title="Game Stats" />
     <CardBody>
       <GameCard game={gameData} />
     </CardBody>
   </Card>
   ```

3. **Use design tokens**:
   ```tsx
   // Use our CSS classes, not inline styles
   <div className="p-4 bg-primary rounded-lg">
     <Button variant="primary" size="lg">
       Play Now
     </Button>
   </div>
   ```
