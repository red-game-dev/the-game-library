## üéØ Design Philosophy

### Gaming-First Aesthetics

Our design system embodies the excitement and energy of crypto gaming:

1. **Electric Purple (#a855f7)** - Web3 identity and innovation
2. **Cyber Lime (#a3e635)** - Success, wins, and rewards
3. **Dark Mode Default** - Optimized for extended gaming sessions
4. **Neon Effects** - Glow effects, glass morphism, animated gradients

### Core Architecture Principles

```css
/* 1. CSS Variables for Everything */
--color-primary: var(--purple-500);
--space-4: 1rem;
--radius-lg: 0.5rem;

/* 2. Utility Classes Using Variables */
.p-4 { padding: var(--space-4); }
.bg-primary { background: var(--color-primary); }
.rounded-lg { border-radius: var(--radius-lg); }

/* 3. Theme Switching via Data Attributes */
[data-theme="dark"] { --color-bg: #030712; }
[data-theme="light"] { --color-bg: #ffffff; }
```

### Spacing Utilities

```css
/* Margin - External spacing */
.m-*   /* All sides */
.mx-*  /* Horizontal (left & right) */
.my-*  /* Vertical (top & bottom) */
.mt-*  /* Top only */
.mr-*  /* Right only */
.mb-*  /* Bottom only */
.ml-*  /* Left only */

/* Padding - Internal spacing */
.p-*   /* All sides */
.px-*  /* Horizontal */
.py-*  /* Vertical */
.pt-*  /* Top only */
.pr-*  /* Right only */
.pb-*  /* Bottom only */
.pl-*  /* Left only */

/* Gap - Flexbox/Grid spacing */
.gap-*    /* All directions */
.gap-x-*  /* Column gap */
.gap-y-*  /* Row gap */
```

// React component

  const setTheme = (theme: string) => {
    document.documentElement.setAttribute('data-theme', theme);
    localStorage.setItem('theme', theme);
  };
  
  return (
    <select onChange={(e) => setTheme(e.target.value)}>
      <option value="dark">Dark</option>
      <option value="light">Light</option>
      <option value="neon">Neon</option>

## ‚ö° Performance Guidelines

### Mobile Optimizations (< 768px)
- Disable complex animations
- Reduce glow effects
- Simplify gradients
- Use transform/opacity only

### Tablet Optimizations (768px - 1024px)
- Selective animations
- Reduced particle effects
- Optimized shadows

### Desktop Features (> 1024px)
- Full animations enabled
- Complex visual effects
- Background decorations
- Parallax scrolling

## üîÑ Migration Path to Tailwind

Our custom system is **100% compatible** with future Tailwind adoption:

1. **Same naming conventions** - All classes follow Tailwind patterns
2. **CSS variables ready** - Can coexist with Tailwind config
3. **No breaking changes** - Add Tailwind without refactoring
4. **Gradual migration** - Both systems work together

```javascript
// Future tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        // Use our existing CSS variables
        primary: 'var(--color-primary)',
        surface: 'var(--color-surface)',
      }
    }
  }
}
```
