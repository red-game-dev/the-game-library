# The Game Library Design System

A modern crypto-gaming platform design system that combines casino excitement with Web3 aesthetics, 
professional trust with gaming delight, and accessibility-first development.

## ğŸ¯ Design Philosophy

### Core Principles

1. **Crypto-Native Aesthetics**
   - Electric purple represents Web3 culture and innovation
   - Cyber lime for success, wins, and positive actions
   - Gradients and glows for modern gaming excitement

2. **Accessibility First**
   - WCAG AAA target (7:1 contrast ratio)
   - 44px minimum touch targets
   - Reduced motion support
   - Screen reader optimized

3. **Performance Optimized**
   - GPU-accelerated animations
   - CSS-based effects over JavaScript
   - Lazy loading and progressive enhancement

4. **Dark Mode Primary**
   - Gaming happens in low-light environments
   - Reduces eye strain during extended sessions
   - Better battery life on OLED screens

### Conditional Styling

```tsx
import { cn } from '@/lib/design-system';

function GameCard({ game, isActive, size }) {
  return (
    <div className={cn(
      'card',                           // Base class
      isActive && 'card-interactive',   // Conditional
      size === 'large' && 'p-8',        // Size variant
      game.isLive && 'glow-live'        // Game state
    )}>
      <h3 className="text-lg font-bold">{game.title}</h3>
      <button className="btn btn-primary w-full">
        Play Now
      </button>
    </div>
  );
}
```
